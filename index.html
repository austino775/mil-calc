Mil‑Calc

Distance Milling Calculator for iPhone

No more slide‑rules. Fast, tactical ranging and mill conversion — built for the field.

How it works

1
Measure the target: Use your reticle or reference target size in inches/centimeters.
2
Enter size & mils: Input the target size and mil reading — Mil‑Calc calculates the range instantly.
3
Apply corrections: Add target height, unit choice (yards/meters), and the app shows holdovers and conversions.
Fast
Instant calculations, low battery usage.
Accurate
MIL↔MOA conversions, ranging formulas you can trust.
Portable
Designed for iPhone; works offline.

Step-by-step directions

  1. Open Mil‑Calc — select either MIL or MOA mode from the main menu.
  2. Choose units — pick inches/centimeters and yards/meters for output.
  3. Enter target size — type the known size (e.g., 18 in for a target) and the measured mils.
  4. Read range — the app displays distance and suggested holdover conversions.
  5. Save or share — export results, copy to clipboard, or email support if you need help.
